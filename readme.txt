■このソフトについて
  TabTextEditorはシンプルなテキストエディターです。

  次の特長があります。

　１．タブブラウザの様に１つのウィンドウでテキストファイルを複数管理できる
　２．テキストファイルを暗号化して見られたくないファイルを守る
　３．よく使うファイルのリンクを登録できるお気に入り機能
　４．わからない単語をカーソルでなぞってWebで検索できるなぞり検索機能搭載
　５．HTMLなどのカラーリングやフォントの変更に対応

■注意事項
　同梱しているフォルダおよびフォルダ内のファイルは一切削除しないでください。
　動作不良の原因となります。

■バージョンアップ
  解凍したファイルを全て既存のファイルに上書きしてください。

■設定の初期化
　初回起動時に作成される、settings.iniを削除すると設定を初期化できます。

■アンインストール
  ダウンロードしたファイルを全て削除してください。

■レジストリについて
  レジストリは一切いじっていません。

■操作方法など
　ソフトウェア動作時に、ヘルプアイコンをクリックするか、
　F1キーを押すことでマニュアルが表示されます。
